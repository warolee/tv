# Contributing

## Git workflow (warolee/tv)

- Work on **`main`**, commit locally, and push with **`git push origin main`**.
- Do **not** create `cursor/...` feature branches or open pull requests unless a maintainer explicitly asks for that workflow in an issue or chat message.

Agent automation (e.g. Cursor Cloud Agents) should follow the same policy; see **`.cursorrules`** in the repo root for Cursor-specific wording.
